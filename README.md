## 🏛️ Estructura Interna de cada Año

Dentro de la carpeta de cada año académico, encontrarás la siguiente organización:

1. **`proyectos/`**: Desarrollo técnico, planos, código o informes de los proyectos ejecutados.
2. **`labor_social/`**: Evidencias, bitácoras e impacto de las actividades con la comunidad.
3. **`investigaciones/`**: Avances de semilleros, tesis y proyectos de investigación internos.
4. **`cronograma/`**: Planificación del tiempo, hitos y metas alcanzadas en ese periodo.
5. **`articulos/`**: Papers, pósteres académicos y publicaciones científicas derivadas.
